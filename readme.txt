钢琴房预约：
每个人能预约当天和第二天的。
预约时需要写上联系方式。
钢琴分A,B两个，预约表格如下：
可预约的时间段为
	工作日：13:00~22:30
	周六周日：9:00~22:30
每个学号每天最多预约两个小时，且预约时间起点为整点或半点，并且以半个小时为单位。
预约成功后，将表格上该位置变颜色，并注明预约人的学号后五位，与预约人名字最后一个字。
遇到假期等还有不开放的时候，需在网上提前注明。
到后需要到办公室押卡，并在网站上注明已到。
若提前离开，则将网站上该时间段改为可预约。
预约时间开始之后半小时未到则取消其预约，并记录在案，多次后禁止其使用一段时间。
可自行网上取消预约。取消预约需在预约时间两个小时之前。否则记录在案，多次后禁止使用一段时间。
需要将每天的钢琴使用情况保留，用于统计钢琴使用情况，并且可以计算任意连续几天内的钢琴使用时间。
